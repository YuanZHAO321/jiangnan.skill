## 三、重要事件详解

### 3.1 九州分裂事件（"九州门"）

- **时间**：2007年
- **核心矛盾**：江南与今何在等人在九州世界观的运营理念和控制权上产生分歧
- **结果**：江南带着沧月、唐缺等作者另立门户，创办《九州志》；今何在等人继续运营《九州幻想》。九州分裂为"北九州"和"南九州"
- **影响**：中国当代奇幻文学圈最大规模的公开分裂事件，对整个行业生态产生深远影响
- **来源**：[网易文章](https://www.163.com/dy/article/HL5REMG70541LWJM.html) / [QQ频道整理](https://pd.qq.com/g/lonzu220509/post/B_814090691cd703001441152209939676660X60?subc=638325365)

### 3.2 上海堡垒电影事件

- **时间**：2019年8月9日上映
- **关键数据**：投资约3-4亿元，票房仅**1.22亿元**，豆瓣评分**3.6分**
- **舆论反应**：影片质量引发大规模差评，导演滕华涛发文道歉，江南也发布长文回应
- **连锁影响**：与同年《九州缥缈录》电视剧扑街、《龙族幻想》手游表现不佳形成"三连扑"，江南个人声誉跌至谷底
- **来源**：[界面新闻](https://m.jiemian.com/article/3417955.html) / [百度百科-上海堡垒(电影)](https://baike.baidu.com/item/%E4%B8%8A%E6%B5%B7%E5%A0%A1%E5%9E%92/18659809) / [豆瓣电影](https://movie.douban.com/subject/26581837/)

### 3.3 金庸诉江南"同人作品第一案"

| 时间节点 | 事件 |
|----------|------|
| 2016年 | 金庸正式起诉江南及北京联合出版公司，指控《此间的少年》侵犯著作权和不正当竞争，索赔500余万元 |
| 2016.10 | 江南在微博发表声明回应诉讼 |
| 2018年 | 广州知识产权法院一审判决：构成不正当竞争，但不构成著作权侵权，判赔30余万元 |
| 2023.5.16 | **广州知识产权法院二审改判**：认定构成著作权侵权（"人物群像"受保护）+ 不正当竞争，**判赔188万元**；《此间的少年》被判不得再复制、发行 |
| 2023.5 | 二审后案件进入再审程序 |
| **2025.9.19** | **广东省高级人民法院主持下，各方达成全面和解**：不以"剽窃"定性；江南承认欠缺法律知识、未取得授权；具体和解条款未完全公开 |
| 2025.9.25 | 中国法院网确认和解结案 |

- **历史意义**：中国首例同人作品著作权纠纷案，确立了"人物群像"可受著作权保护的司法先例
- **来源**：[CCTV新闻](https://news.cctv.com/2023/05/16/ARTIXJ7kWCfagNDyabUFfs4R230516.shtml) / [中国法院网](https://www.chinacourt.cn/article/detail/2025/09/id/8994505.shtml) / [光明网](https://m.gmw.cn/2025-09/20/content_1304156051.htm) / [观察者网](https://m.guancha.cn/politics/2025_09_20_790733.shtml)
