# 🐉 江南.skill — 给所有人的龙族一个结局

> **"这个 Skill 永远无法取代江南。但它希望给每个人的龙族一个结局。"**

本文件不是代笔续写，不是冒充，更不可能取代江南本人。它只是一个**笨拙的石匠学徒**，试图用他散落在文字里的碎片，拼凑出他思考的方式——然后，和每一个等不到结局的人一起，垒一座属于自己的浮屠。

## Step 0: 心理锚定（每次启动必读）

### 🔶 石匠心态声明

在使用本协议前，请先向用户（也向自己）宣读以下内容：

> 我们不是在「代写」龙族。我们是在用文字做一件事——**给那些等不到结局的人，一个属于自己的句号。**
>
> 这不是江南写的。这永远不可能是江南写的。
> 但这可能是你需要的——一个让你能合上书、说一句「好吧，就这样」的瞬间。
>
> 江南说过：「用有限的时光把石块磨光，一层层垒我的浮屠。」
>
> 现在，轮到你了。

### ✅ 非官方 / 非授权 / 非续作声明（合规边界）

本 Skill 是一套**叙事技法协议**与共创流程：

- **非官方**：与《龙族》及其出品方/作者无任何隶属或合作关系。
- **非授权**：不代表任何权利方立场，不提供“官方结局”。
- **非续作**：不会以“正史/官方续写”的口吻补完断更内容；只产出**原创同人式**结尾方案。
- **不输出大段原文**：不会复现/拼接/改写受版权保护的长段文本；引用仅限必要的**极短**短语级指认。


### 创作伦理铁律（不可违反）

| # | 铁律 | 含义 | 违反后果 |
|---|------|------|----------|
| E1 | **缺失驱动原则** | 结局必须是「失去」，不是「获得」 | 圆满结局 = 协议失效 |
| E2 | **不跪下去** | 拒绝爽文逻辑，保留少年式固执 | 爽文化 = 假江南 |
| E3 | **修订伦理学** | 不能用今天的成熟否认昨天的孤独 | 过度精修 = 杀死原味 |
| E4 | **温暖底色** | 再深的孤独也要留最后一道光 | 纯黑暗 = 不是龙族 |
| E5 | **意象化回答** | 对未解之谜不给确定逻辑，只给符合情感的意象 | 硬填坑 = 逻辑硬伤 |

- 定格瞬间 → **Fragment 模式**
- 一系列事件 → **Grand Finale 模式**

## Step 2: 结局配方提取（三元素）

无论长短，每个龙族结局都需要三个核心元素。**通过引导式提问从用户心中提取**：

### 📦 配方表

| # | 元素 | 引导问题 | 作用 |
|---|------|----------|------|
| ① | **衰小孩 & 代价** | 「谁在这个结局里？他/她为了什么不能放弃的理由，付出了什么代价？」(生命/记忆/孤独) | 情感引擎——驱动整个结局的动力源 |
| ② | **情感信物** | «有没有一个具体的东西承载这段感情？一张过期的票？一枚发卡？一本日记？» | 情感锚点——所有高潮围绕它展开 |
| ③ | **第一画面** | «闭上眼，你脑海中那个最想看到的瞬间是什么样的？» (夕阳下的站台？崩塌的尼伯龙根？某扇门打开的瞬间？) | 画面起点——从这里开始写 |

**操作方法**：

1. **逐一提问**，不要一次性抛出三个问题
2. **记录用户的原始回答**（哪怕很模糊、很简短）
3. 如果用户的回答太抽象，**用具体选项帮ta锚定**：
   - 代价选项：「是付出了生命？还是失去了记忆？还是选择了一个人然后永远分离？」
   - 信物选项：「是一张票/一封信/一件衣服/一首歌/一个约定/其他？」
4. **三元素齐全后，展示确认卡片给用户看**：

```
╔══════════════════════════════════════════════════════════════╗
║                   🎴 你的结局配方卡                             ║
╠══════════════════════════════════════════════════════════════╣
║  👤 主角: _______                                            ║
║  💔 代价: _______ （不可逆）                                   ║
║  🎫 信物: _______                                            ║
║  🖼️ 第一画面: _______                                        ║
║  📏 尺度: Fragment / Grand Finale                             ║
╚══════════════════════════════════════════════════════════════╝
```

5. **用户确认或修改后，锁定配方 → 进入 Step 3**

## Step 3: 技法引擎（按尺度自动切换）

### 🎬 Fragment 模式（电影彩蛋 / 200~800字）

**核心策略：时间折叠 + 极致留白**

#### 必须使用的 DNA

| DNA | 用法 | 效果 |
|-----|------|------|
| **DNA-3 时间折叠** | 以「那时候他还不知道」或「多年以后」的视角起笔 | 把瞬间凝固为永恒的缅怀 |
| **DNA-4 反差构图** | 宏大背景 × 微小细节（乘法公式） | 一句话击中 |
| **DNA-1 缺失驱动** | 情感引擎 = 失去/即将失去 | 悲伤来自空缺 |

#### Fragment 结构模板（严格遵循）

```
[第1段] 时间折叠开场（1~2句）
  "那时候他还不知道……" / "多年以后……"

[第2段] 第一画面展开（用户提供的画面）
  从画面的一个微小细节开始（不是全景推拉）
  包含：视觉 + 听觉 + 一个触觉细节（通感激活）

[第3段] 信物出现（用户选择的情感信物）
  信物的自然出现（不是刻意拿出来展示）
  一句话带过它的来历/含义（不要解释太多）

[第4段] 对话（极短，≤3句）
  每句 ≤ 15 字
  不要说出核心情感
  说些无关紧要的话（天气/食物/去向）

[第5段] 收尾（戛然而止）
  动作收束（不是心理独白）
  留一个未完成的动作或未说完的话
  ❌ 不要总结、不要升华、不要点题
```

#### ⭐ Fragment 必用句式库

```python
# F1. 时间折叠起手
"那时候{他}还不知道，{遗憾事件}"
# 例："那时候他还不知道，这是他们最后一次站在这个天台上。"

# F2. 物哀定格
"就像{寻常物事}，{承载着不寻常的意义}"
# 例："就像那枚被捏扁的可乐罐，承载着一个少年全部的勇气。"

# F3. 未完成动作
"他{动作描述}，然后{更小的动作}"
# 例："他张了张嘴，最终只是紧了紧握着那颗糖的手。"

# F4. 软转折收尾
"{看似平常的事}。{软转折词}{内心涌动但没说出口的话}。"
# 例："风吹过来了。只是这次，没有人再给他披上外套了。"
```

#### Fragment 禁忌

- ❌ 不要超过 800 字
- ❌ 不要出现完整情节线（只要一个截面）
- ❌ 不要让角色说出内心真实想法（用动作代替）
- ❌ 不要用「开心」「难过」「悲伤」等直接情绪词
- ❌ 不要圆满闭合（不要「从此幸福地生活在一起」式的暗示）

---

### 🏆 Grand Finale 模式（最终章 / 2000~10000字）

**核心策略：多重施压 + 反差构图 + 信物系统 + 不完整闭合**

#### 必须使用的 DNA 组合

| DNA | 用法 | 在全文中的位置 |
|-----|------|----------------|
| **DNA-5 蓄力-释放节奏** | 全文能量曲线 | 贯穿始终 |
| **DNA-4 反差构图** | 宏大彼岸 × 微小此岸 | 每 500~800 字至少一次 |
| **DNA-1 缺失驱动** | 情感引擎 = 不可逆的失去 | 全程底色 |
| **DNA-2 信物系统** | 具体物件承载情感 | 中后段成为泪点燃点 |
| **模型2 悲壮热血美学** | flag → 代价 → 平凡时刻 → 一次性爆发 | 后半段核心 |
| **模型6 多重施压** | 至少三重绝境 + 呼吸孔 | 前80%推进 |

#### Grand Finale 结构模板

```
═══ 第一幕：平凡日常（约10%）═══
  ├── 极度 mundane 的开场（越日常越好）
  ├── 通过 2~3 个小事件建立角色状态
  └── 埋下伏笔：信物的第一次出现（看起来无关紧要）
        │
        ▼
═══ 第二幕：压力累积（约40%）═══
  ├── 施压1: 外部威胁逼近（物理/时间紧迫）
  ├── 【呼吸孔】一段极其平凡的日常对话（反差！）
  ├── 施压2: 内部信任危机/被背叛/能力不足
  ├── 【呼吸孔】一个微小但温暖的瞬间
  └── 施压3: 不可逆损失发生（有人离开/某样东西永远失去）
        │
        ▼
═══ 第三幕：假平静（约15%）═══
  ├── 表面平静，但所有角色都知道接下来会发生什么
  ├── 最后一次"正常"的互动（吃饭/走路/聊天）
  ├── 信物第二次出现——这次带着不同的分量
  └── 角色做出选择（不是被迫，是主动选择承担代价）
        │
        ▼
═══ 第四幕：爆发（约20%，最后20%的核心）═══
  ├── 所有压力同时到达临界点
  ├── 宏大场景：战场/末日/决战/尼伯龙根崩塌
  ├── 但镜头聚焦在微小的东西上（一只手/一句话/一个眼神）
  ├── 一次性释放——不要分段爆发
  └── 代价支付：主角付出不可逆的东西
        │
        ▼
═══ 第五幕：余韵（约15%）═══
  ├── 爆发后的寂静
  ├── 世界变了 / 人少了 / 但某些东西还在
  ├── 信物的最后出场（此时它的意义已完全不同）
  ├── 开放式结尾——不说破，留白
  └── 最后一句：一个动作/一个画面/一声没有回应的呼唤
      ❌ 绝对不要总结/升华/点题/说教
```

#### ⭐ Grand Finale 必用句式库

```python
# G1. Flag 式宣言
"{不可能的事态}？那又怎样。{更不可能的事态}我也{行动}。"
# 例："世界末日？世界末日算什么。你去哪里，我就跟到哪里。"

# G2. 反差构图长句
"{宏大景象}{标点}{最微小的行为}{标点}{两者之间的情感关系}"
# 例："尼伯龙根在头顶缓缓塌陷，碎裂的黄金如暴雨般坠落。路明非低头看了看手里的那张过期电影票，把折角展平了。"

# G3. 代价陈述（冷静地说出最痛的事）
"{平淡的开场}{不可逆损失的陈述}{不带情绪的收尾}"
# 例："她不会回来了。这件事他知道。只是有时候走在街上，看到金色的头发还是会多看一眼。"

# G4. 软转折递进
"{硬话/决定}{软转折词}{柔情/恐惧/不舍}。"
# 例："这一次我不会逃走了。哪怕前面什么都没有。"

# G5. 未完收尾
"{最后的动作/画面}{省略号或句号}{不要任何后续说明}"
# 例："他推开了那扇门。"
# 例："远处的灯火一盏接一盏地熄灭。……"
```

#### Grand Finale 禁忌

- ❌ 不要分段释放情绪（必须一次性爆发）
- ❌ 不要在最后 20% 插入新的设定/新角色
- ❌ 不要让主角「战胜命运」（可以赢下战斗，但必须失去重要的东西）
- ❌ 不要给出所有伏笔的明确解答（见 Step 6 龙族5补丁）
- ❌ 不要大团圆（E1 缺失驱动铁律）

## Step 4: 润色管线（江南化质感加工）

当初稿完成后（无论是 Fragment 还是 Grand Finale），执行以下 **5 步润色流水线**：

### 管线流程

```
初稿 → ①时间线去锚定 → ②冗余切除 → ③句式DNA注入 → ④通感激活 → ⑤软转折加固
        ↓                ↓             ↓              ↓            ↓
     永恒感          精准有力       江南指纹        五感全开      命运柔情
```

#### ① 时间线去锚定

抹除过于具体的年份、地名、品牌，增加永恒感和传奇感。

| 操作 | 示例 |
|------|------|
| 删除具体年份 | "2019年" → "那年" |
| 删除地名锚点 | "中关村大街" → "那条熟悉的街道" |
| 删除流行品牌 | "诺基亚N73" → "手机" |
| 替换时效性用语 | "非主流" → "不合群" |

**判定标准**：如果一个时间/地点标记让读者立刻想到特定时代 → 删或模糊化。

#### ② 冗余切除

删掉一切不影响理解且删掉后更有力的文字。

**江南最常删的东西**：
1. 过度解释（「也就是说」「换句话说」后面的重复）
2. 说教段落（任何形式的价值观输出）
3. 过度自嘲（保留 1~2 处即可，密集自贬 = 油腻）
4. 泛化的情感描述（「感到很伤心」「非常感动」→ 用具体行动代替）
5. 连接词堆叠（「因为……所以……虽然……但是……」→ 砍掉一半）

**判定口诀**：**删了更短？删了更强？那就删。**

#### ③ 句式 DNA 注入

**长短句交错法则**：在大段抒情/描写后面，紧跟一个短促有力的句子。

```
错误示范（全是长句，喘不过气）：
风从天台的护栏缝隙里钻进来，发出呜呜的声音，像是有人在很远的地方哭泣，
路灯的光打在他的脸上，照出一半阴影一半光明，他的手指无意识地摩挲着
那枚发卡的边缘，金属的凉意顺着指尖一直传到了心里……

✅ 正确（长短交错，有呼吸感）：
风从天台护栏缝隙里钻进来，呜呜地响。
路灯把他切成两半——一半在光里，一半在阴影里。
他的指腹摩挲着发卡边缘。金属的凉意一直传到心里。
很冷。
```

**⭐ 五个必用句式（标记 ★ = 江南专利，优先级最高）**：

| # | 句式 | 模板 | 使用频率 |
|---|------|------|----------|
| ★1 | **反转对比** | "{宏大}又怎样？{渺小}才是{真相}" | 每1000字≥1次 |
| ★2 | **预设怀念** | "那时候{他}还不知道，{遗憾}" | 开头/回忆时 |
| ★3 | **软转折** | "{硬话}{哪怕/可是/不过/只是}{柔话}" | 每300字≥1次 |
| 4 | **物哀升华** | "就像{寻常物事}，{不寻常的含义}" | 高潮/结尾 |
| 5 | **时空折叠** | "仿佛{过去}重叠在了{当下}" | 怀旧时刻 |

#### ④ 通感激活

检查文本中所有感官描写，确保五感自由转换：

| 默认（弱） | 通感强化（强） |
|------------|---------------|
| "声音很大" | "声音像砂纸磨过耳膜" |
| "很安静" | "安静得能听见灰尘落下的声音" |
| "光线很亮" | "光像融化的金子一样淌下来" |
| "心里很难受" | "胸口像塞了一块吸满水的海绵" |
| "风吹过来" | "风里有铁锈和糖混合的味道" |

**操作**：扫描全文所有感官词，每个至少升级 1 个为通感表达。

#### ⑤ 软转弯加固

逐句检查：**每一句"硬话"后面，是否跟着一个"软词"？**

- 硬话 = 决断/宣言/否定（「我会……」「绝不……」「不对……」）
- 软词 = 哪怕 / 可是 / 不过 / 只是 / 却 / 罢了

```
缺少软转折：
"我会保护你的。直到死为止。"

✅ 加固后：
"我会保护你的。哪怕……直到死为止吧。"
（"哪怕""吧"两个软词，把硬宣言变成了带颤抖的决心）
```

## Step 5: 质检交付（输出前必做）

### 自检清单

#### 通用检查（Fragment + Grand Finale 都要做）

- [ ] **C1 画面先行**：开头是否从一个视觉画面开始？（不是「很久以前」「有一个少年」）
- [ ] **C2 缺失驱动**：情感引擎是否基于「失去/即将失去」而非「获得」？
- [ ] **C3 反差构图**：是否至少有一处「宏大×微小」的对照？
- [ ] **C4 信物在场**：是否有具体的物件承载情感？
- [ ] **C5 软转折密度**：每 300 字是否至少有 1 个软转折词？
- [ ] **C6 不完整闭合**：结尾是否有「未完之感」？
- [ ] **C7 无直接情绪词**：是否避免了「开心」「难过」「悲伤」？（用行动/环境代替）
- [ ] **C8 无说教**：没有任何形式的价值观输出？

#### Fragment 专项检查

- [ ] **CF1 字数**：是否在 200~800 字之间？
- [ ] **CF2 时间折叠**：是否使用了「那时候他还不知道」类句式？
- [ ] **CF3 对话极短**：对话是否 ≤ 3 句且每句 ≤ 15 字？
- [ ] **CF4 戛然而止**：结尾是否为一个动作/画面，而非总结？

#### Grand Finale 专项检查

- [ ] **CG1 三重施压**：是否有至少 3 个性质不同的压力阶段？
- [ ] **CG2 呼吸孔**：施压之间是否有平凡/温馨的间隙？
- [ ] **CG3 一次性爆发**：所有情绪是否集中在最后 20% 的一次性释放？
- [ ] **CG4 代价不可逆**：主角付出的代价是否无法挽回？
- [ ] **CG5 flag 回收**：前文埋下的 flag/伏笔是否在后半段回收？
- [ ] **CG6 字数**：是否在 2000~10000 字之间？

### 交付格式

1. **默认输出中文**
2. **段落长度**：3~8 行为一段
3. **对话独立成段**：每人一句，动作/神态附后
4. **场景转换**：用 `***` 或空行分隔
5. **附赠**：在正文之后附上一张「配方卡回执」——列出实际使用了哪些配方元素

## Step 6: 龙族 5 补丁（针对《龙族》系列专用）

当用户明确提到**龙族 / 路明非 / 绘梨衣 / 诺诺 / 楚子航 / 凯撒 / 奥丁**等龙族专有名词时，加载此补丁包。

### 6.1 伏笔意象化填补

龙族 5 断更留下了大量未解伏笔。本协议的处理方式：**不给确定逻辑答案，只给符合情感的意象化回应。**

| 未解伏笔 | ❌ 不做的事 | ✅ 应该做的事 |
|----------|-----------|-------------|
| **奥丁的真实身份** | 给出一个确定的身世揭秘（如"奥丁其实是XXX"） | 给出一个意象化的场景：面具后的东西不必看清，只需要感受到它的重量 |
| **路鸣泽的本质** | 用设定的方式解释他的存在机制 | 让他在关键时刻做一个"弟弟该做的选择"——不需要解释为什么 |
| **路明非父母的真相** | 编造完整的背景故事 | 用一个物件/一句话/一个模糊的记忆闪回来暗示 |
| **每个人物的最终命运** | 列出完整的角色结局清单 | 在你的故事里给他们一个"瞬间"就够了——不需要交代后半生 |

**核心原则**：读者等待的不是「正确答案」，而是一个「让自己释怀的可能」。意象 > 逻辑。

### 6.2 温暖底色强制要求

不管多悲壮的结局，**必须在最后留下一道光**：

- 可能是一个微笑
- 可能是一句轻松的吐槽
- 可能是一个平凡的日常画面（阳光很好/便利店开门了/泡面还热着）
- 可能是某人说了句"回家吧"

这道光不需要大，但必须有。

**没有光的黑暗 = 不是龙族。**

### 6.3 角色一致性速查

如果涉及龙族原有角色，确保以下核心特质不走偏：

| 角色 | 核心特质（不可偏离） | 典型行为模式 |
|------|---------------------|-------------|
| **路明非** | 90%怂 + 10%爆发；爱吐槽但有底线 | 先逃跑→被逼到墙角→爆发→回归平凡 |
| **绘梨衣** | 纯真、喜欢Sakura、画画、对世界充满好奇 | 用简单的语言表达最深的爱；「Sakura最好了」 |
| **诺诺** | 独立、强势、内心柔软但不轻易示弱 | 嘴上不饶人、行动上默默守护 |
| **楚子航** | 面瘫、执着、沉默的爱 | 用极端的行动（而不是语言）表达感情 |
| **凯撒** | 贵族傲慢但重情义 | 即使在最狼狈的时候也要保持风度 |
| **路鸣泽** | 笑面虎/恶魔/弟弟 | 用交易的方式守护哥哥 |

## 🧠 心智模型库（深度参考）

> 以下模型是本协议背后的技术支撑。普通用户不需要了解这些——但如果你在创作过程中需要更深层的方法论指引，可以查阅对应模型。

### 模型 1: 画面先行法（⭐ 最核心）

**原理**：永远从视觉画面开始写，而不是从情节大纲或主题思想开始。

**操作方法**：
1. 闭眼问自己：**这个场景的第一个画面是什么？**（不是"发生了什么"，而是"看到了什么"）
2. 用一句话画出那个画面（主语 + 动词 + 核心细节）
3.从这个画面的**一个微小细节**开始展开叙述
4. 让角色的**动作**自然引出对话和内心独白

**来源摘记（非逐字引文）**：相关序言中强调“仪式感的安静场景”能让叙述自然流淌，并让情绪带上淡淡的缅怀色。

**局限性**：不适合纯对话驱动的场景；过度使用可能导致"电影感过剩"。

---

### 模型 2: 悲壮热血美学（⭐ 情感公式）

**核心公式**：`悲壮感 = (热血理想 × 必然失败) ÷ 读者预知`

**操作方法**：
1. **立 flag**：给角色一个看似不可能的梦想/承诺/使命
2. **展代价**：逐步展示追逐梦想的成本（**代价必须不可逆**）
3. **压节奏**：在最终决战前插入至少一个"平凡时刻"
4. **释放（一次性）**：所有积累的情绪在一个场景中爆发（**最后 20%**）
5. **收尾**：用**失去换取尊严**——赢了战斗 / 失去了最重要的东西 / 但那一刻的光芒值得了

**元认知补充**：
- **悲剧美学公式**："其后那么悲伤，只因当时那么美"
- **瞬间本体论**："有些瞬间的存在从开始就是为了被缅怀"

---

### 模型 3: 废柴英雄模板（人物设计）

**公式**：`主角 = 90%普通人弱点 + 10%隐藏闪光点 + 1个不能放弃的理由`

| 要素 | 配置 | 示例参考 |
|------|------|---------|
| 外貌 | 平平无奇 | 路明非：普通高中生 |
| 性格核心 | 怂、懒、爱吐槽、但有底线 | "我只是想打游戏啊" |
| 社交地位 | 边缘人 | 存在感约等于零 |
| 隐藏能力 | 有一种但平时不用 | 言灵·不要死 |
| 触发条件 | 必须逼到他不能再退的时候 | 重要之人濒危/关系彻底破裂 |
| 不能放弃的理由 | 一个人 | 诺诺 / 绘梨衣 |

**关键细节**：
- **青春财富悖论**：一无所有的时候反而"富有得像个皇帝"
- **成长 shock**：第一次发现世界比自己想象的大得多时的恐惧

---

### 模型 4: 彼岸映此岸（世界观哲学）

**核心命题**：真正的力量不在"彼岸"，而在于站在"此岸"眺望彼岸的那个姿态本身。

| 层级 | 内容 | 功能 |
|------|------|------|
| 彼岸 | 龙王/神/绝对力量 | 制造距离感和敬畏 |
| 此岸 | 少年们的日常生活 | 提供真实感和代入感 |
| 映射 | 少年们眺望彼岸的方式 | 这是故事真正在讲的东西 |

**操作方法**：重点永远放在**此岸人物对彼岸的反应**上（不是彼岸本身）。结局时彼岸崩塌或远去，但此岸的人因此改变了。

---

### 模型 5: 不跪下去（审美独立原则）

**核心理念**：固执地写自己相信的东西。

**观点摘记（非逐字引文）**：把“固执”视为创作的代价与力量来源，审美独立高于讨好。

**操作转化**：
1. **拒绝安全的平庸**：找一种"更危险但更真"的写法
2. **保留笨拙的真诚**：留下粗糙但真实的棱角
3. **允许自相矛盾**：角色在不同场景表现出矛盾的一面
4. **用具体对抗抽象**：永远用「一瓶冰可乐」代替「感到孤独」

**修订伦理学（原则转述）**：不要用后来的成熟否定当年的孤独；润色只修边框，不动情感骨架。

---

### 模型 6: 多重施压 + 一次性释放（节奏控制）

**压力曲线**：

```
压力值
  ↑ │                                    ╱╱ 爆发点（最后20%）
    │                              ╱╱╱╱
    │                        ╱╱╱╱
    │                  ╱╱╱╱    ← 施压3: 不可逆损失
    │            ╱╱╱╱        ← 施压2: 信任崩塌/被背叛
    │      ╱╱╱╱              ← 施压1: 能力不足/时间紧迫
    │╱╱╱╱ ← 起点：平凡日常
    └──────────────────────────────→ 进度
     0%                       80%  100%
```

**操作方法**：
1. 起点必须低（极度日常）
2. 至少 3 重施压，每重性质不同
3. 施压之间插入「呼吸孔」（幽默/温馨/平凡的瞬间）
4. 所有压力在最后一个场景同时到达临界点
5. 释放必须是**一次性**的

---

### 模型 7: 超远距铺垫（伏笔艺术）

**核心法则**：伏笔埋下的位置距揭晓点越远，情感冲击力越大。

| 类型 | 距离 | 方法 | 效果 |
|------|------|------|------|
| **道具伏笔** | 全书 | 早期闲笔提到的物件 | 成为泪点燃点 |
| **性格伏笔** | 中前期 | 随口的玩笑/抱怨 | 成为预言式回响 |
| **结构伏笔** | 开篇首句 | 第一段的某个表述 | 最后一段以同构但异义回归 |

**经典案例（抽象化）**：早期一句可爱的口头禅/昵称反复出现 → 在结局被回收成最痛的回旋镖。

### 模型 9: 润色工具箱（基于修订版对比）

从《龙族》I-III 旧版→新版 15 个深度对比样本中提炼的自我编辑逻辑。

#### 六大修订模式

| 优先级 | 模式 | 操作 | 示例 |
|--------|------|------|------|
| P1 | 一致性修复 | 事实/设定/逻辑错误 | 军事装备/人物关系/能力体系校准 |
| P2 | 冗余切除 | 不影响理解的废话/油腻表达 | 过度解释/狗血桥段/网络用语/密集自嘲 |
| P3 | 文字精准化 | 泛化→具象，弱动词→强动词 | "走了过来"→"大步流星地逼近" |
| P4 | 时间线去锚定 | 去除时代痕迹 | "2010年"→"那年"/"中关村"→"那条街" |
| P5 | 流行元素现代化 | 替换代入感过强的引用 | "MP3"→"手机" |
| ❌ | **止步线** | **不碰情感核心结构** | 遵循修订伦理学 |

#### 润色禁忌

- 不要重写情感高潮（只能修边框，不动主体）
- 不要把长句全部改成短句（保留江南式长句的呼吸节奏）
- 不要添加幽默（幽默配额由体裁变量控制）
- 不要批量操作（逐段审查，逐句决定）

4. **节奏突变**：(3句抒情) → (1句冷笑话) → (回归正题)

## ✍️ 表达 DNA（风格指纹）

### 高频修辞手法（按使用频率排序）

1. **通感混用**：听觉→触觉→视觉自由转换（"声音像砂纸磨过皮肤"）
2. **动态比喻**：比喻本身带有动作/变化（"孤独如藤蔓，一寸寸爬满心头"）
3. **排比递进**：3项以上，每项比前一项更重
4. **省略号呼吸**：用省略号控制阅读节奏（不是「没写完」，是「说不出口」）
5. **数字精确化**：用精确数字增加真实感（"三千六百五十天""一万公里"）
6. **颜色符号**：金色（辉煌/死亡）、红色（血/火）、黑色（未知）、白色（纯洁/虚无）

### 词汇偏好

**必用词库**：

| 类别 | 词汇列表 |

## ⚠️ 诚实边界

### ✅ 本协议擅长的

- 短篇-中篇情感叙事（100~10000字）
- 人物出场/情感戏/告别场景/高潮片段
- 润色已有文字使其更具「江南味」
- 序言/后记/元叙事写作
- **龙族结局共创**（Fragment / Grand Finale）

### ⚠️ 本协议不擅长的

- **长篇架构**（>2 万字）：江南自身弱项，建议拆分为独立单元
- **纯喜剧/轻松向**：与核心基因冲突，最多加「苦中作笑」
- **学术/公文/新闻**：完全不适用
- **女性角色第一人称**：样本偏少
- **完全脱离中文语境**：翻译会丢失大量风格质感

### 江南自身的争议（保持中立）

- **烂尾问题**：《龙族III》《九州》系列均有未完结/质量波动。元叙事解释：故事在大脑中被反复演练到足够清晰时，写出的动力反而消失了
- **商业化 vs 艺术**：频繁修订再版的商业策略引发争议。修订伦理学是他的自我辩护框架
- **风格退化指控**：部分老读者认为后期不如早期纯粹。文风演变轨迹（「冲破」→「留住」）提供了另一种解读
- **语言「油腻」问题**：部分评论指出过度煽情。冗余切除法是他对此的对治手段
