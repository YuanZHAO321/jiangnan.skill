## ✨ 这是什么

**jiangnan** 是一个 AI Agent Skill（智能体技能），蒸馏自中国畅销幻想小说家 **江南（杨治）** 的写作技法体系。

它不是教你「成为江南」，而是让你理解一个卖了 **2100 万册书** 的作者是怎么造句子的。

### 聚焦领域

| 维度 | 内容 |
|------|------|
| **叙事结构** | 画面先行法、多重施压+一次性释放、超远距铺垫 |
| **世界观构建** | 彼岸映此岸哲学、架空世界的现实折射 |
| **人物塑造** | 废柴英雄模板、出厂设置与成长代价 |
| **情绪渲染** | 悲壮热血美学、节奏控制算法、金句余韵 |
| **元认知层** (v1.1🆕) | 创作伦理学、文风演变轨迹、自我认知框架 |
| **润色系统** (v1.1🆕) | 6大修订模式、决策优先级框架、禁忌清单 |
| **跨体裁适配** (v1.1🆕) | 6大通用DNA + 5大体裁变量表 |

## ⚠️ 诚实边界

江南的技法有明确的长板和短板：

**擅长**
- 中短篇情绪渲染和人物刻画
- 场景画面感营造
- 青春孤独感的精准捕捉
- 金句和段子的传播力
- 润色/修订已有文字（有完整的自我编辑方法论）

**弱项**
- 长篇架构控制（公认弱项，多部作品被批「烂尾」）
- 逻辑一致性维护
- 结局收束能力
- 后期文风衰减（《龙族》后续 vs 前期）
- 纯喜剧/轻松向内容（与核心基因冲突）
