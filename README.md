## ✨ 这是什么

**jiangnan** 是一个 AI Agent Skill（智能体技能），蒸馏自中国畅销幻想小说家 **江南（杨治）** 的写作技法体系。

它不是教你「成为江南」，而是让你理解一个卖了 **2100 万册书** 的作者是怎么造句子的。

### 聚焦领域

| 维度 | 内容 |
|------|------|
| **叙事结构** | 画面先行法、多重施压+一次性释放、超远距铺垫 |
| **世界观构建** | 彼岸映此岸哲学、架空世界的现实折射 |
| **人物塑造** | 废柴英雄模板、出厂设置与成长代价 |
| **情绪渲染** | 悲壮热血美学、节奏控制算法、金句余韵 |

## 🧠 核心：7 个心智模型

| # | 模型 | 一句话 |
|---|------|--------|
| 1 | **画面先行法** ⭐ 最核心 | 先构思电影级关键画面，再反向串联情节 |
| 2 | **悲壮热血美学** | 热血是爆发，悲伤是归宿，两者之间的张力是灵魂 |
| 3 | **废柴英雄模板** | 主角出厂设置是「衰/怂/孤独」，爆发必须付代价 |
| 4 | **彼岸映此岸** | 架空世界不是逃避现实，而是用距离感折射人性困境 |
| 5 | **不跪下去** | 可以亲近受众但不要跪下去——审美独立高于一切 |
| 6 | **多重施压→一次性释放** | 铺垫欢乐 → 积压不安 → 反转坠落 → 金句余韵 |
| 7 | **超远距铺垫** | 向通向悲剧的路上洒满鲜花——预支悲伤 |

> 完整模型定义、操作方法和局限性说明见 [SKILL.md](SKILL.md)

## ⚠️ 诚实边界

江南的技法有明确的长板和短板：

**擅长**
- 中短篇情绪渲染和人物刻画
- 场景画面感营造
- 青春孤独感的精准捕捉
- 金句和段子的传播力

**弱项**
- 长篇架构控制（公认弱项，多部作品被批「烂尾」）
- 逻辑一致性维护
- 结局收束能力
- 后期文风衰减（《龙族》后续 vs 前期）
