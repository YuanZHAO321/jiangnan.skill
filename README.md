## ✨ 这是什么

**江南.skill** 是一个 AI Agent Skill（智能体技能）——更准确地说，它是一套**「龙族结局共创叙事协议」**。

它**不是**代笔续写，**不是**冒充，更**不可能**取代江南本人。它只是一个笨拙的石匠学徒，试图用他散落在文字里的碎片，拼凑出他思考的方式——然后，和每一个等不到结局的人一起，垒一座属于自己的浮屠。

### 从 v1.x 到 v2.0：一次架构跃迁

| | v1.x「技法手册」 | **v2.0「叙事协议」** |
|--|------------------|---------------------|
| 用户角色 | 发指令的人 | 共同创作的伙伴 |
| 交互方式 | 用户说需求 → AI调技法 | **引导式提问 → 提取创作素材** |
| 输出目标 | 「像江南的文字」 | **「属于你的龙族结局」** |
| 核心隐喻 | 工具箱 / 参考手册 | **石匠的修行 / 浮屠的垒砌** |
| 适用场景 | 通用写作辅助 | **聚焦：给龙族一个结局** |

## 🧠 美学百科

### 心智模型 1-9

| # | 模型 | 原理 / 公式 | 如何使用 | 线索 |
|---|------|-------------|----------|------|
| 1 | 画面先行法 | 以画面而不是主题作为起点；先见画面，再见情节。 | 先写首个镜头，再让动作自然引出对白和内心。 | 相关序言强调安静仪式感能让叙述自然流淌。 |
| 2 | 悲壮热血美学 | `悲壮感 = (热血理想 × 必然失败) ÷ 读者预知` | 先立目标，再铺代价、呼吸孔和一次性爆发。 | 反复出现“当时很美，后来很悲伤”的自述线索。 |
| 3 | 废柴英雄模板 | `主角 = 90%普通人弱点 + 10%隐藏闪光点 + 1个不能放弃的理由` | 把弱点、闪光点和不可放弃的理由绑成驱动。 | “衰小孩”与“平凡人也能被命运推着走”的叙述基调。 |
| 4 | 彼岸映此岸 | 不是直接写彼岸，而是写此岸如何眺望彼岸。 | 重要场景优先写反应、姿态和距离感。 | 架空设定常被处理为现实经验的折射。 |
| 5 | 不跪下去 | 审美独立高于讨好，固执是代价也是力量。 | 先守住审美骨架，再做局部打磨。 | 自述里反复强调“固执”“不要跪下去”。 |
| 6 | 多重施压→释放 | 先累计压力，再一次性释放，避免分段泄压。 | 至少三重压力，最后 20% 集中爆发。 | 作品经常出现“平凡时刻→压迫→爆发”的曲线。 |
| 7 | 超远距铺垫 | 伏笔距离越远，回收时情感冲击越强。 | 把闲笔、道具、口头禅提前埋到更前面。 | “早期一句话，结局变回旋镖”的反复模式。 |
| 8 | 跨体裁通用 DNA | 缺失驱动 / 信物 / 时间折叠 / 反差构图 / 蓄力节奏 / 小兽隐喻。 | 先判断体裁，再切换对应 DNA 组合。 | 不同体裁词汇会变，但底层节奏保持一致。 |
| 9 | 润色工具箱 | 先修硬伤，再删废话，再调精度，最后才碰节奏。 | 先一致性、再冗余、再精确、再去锚定。 | 修订版对比里最常见的是精度与一致性修复。 |

## ⚠️ 诚实边界

### ✅ 协议擅长的

- **龙族结局共创**（Fragment / Grand Finale 双模式）
- 中短篇情感叙事（100~10000字）
- 人物出场/情感戏/告别场景/高潮片段
- 润色已有文字使其更具「江南味」
- 科幻/校园/奇幻/热血多体裁适配

### ⚠️ 协议不擅长的

- **长篇架构**（>2 万字）：江南自身弱项，建议拆分为独立单元
- **纯喜剧/轻松向**：与核心基因冲突，最多加「苦中作笑」
- **学术/公文/新闻**：完全不适用
- **女性角色第一人称**：样本偏少
- **完全脱离中文语境**：翻译会丢失大量风格质感

### 创作伦理铁律（5 条不可违反）

| # | 铁律 | 含义 |
|---|------|------|
| E1 | **缺失驱动** | 结局必须是「失去」，不是「获得」 |
| E2 | **不跪下去** | 拒绝爽文逻辑，保留少年式固执 |
| E3 | **修订伦理学** | 不能用今天的成熟否认昨天的孤独 |
| E4 | **温暖底色** | 再深的孤独也要留最后一道光 |
| E5 | **意象化回答** | 对未解之谜不给确定逻辑，只给情感意象 |
